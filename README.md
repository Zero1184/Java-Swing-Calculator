# Java Calculator

A simple **calculator application** built with Java Swing. It supports basic arithmetic operations, square roots, and squaring numbers.

## Features

- Addition (`+`), subtraction (`-`), multiplication (`x`), division (`÷`)
- Square root (`√`) and square (`X²`)
- Clear (`C`) and Clear Entry (`CE`) buttons
- Decimal and negative numbers support
- Basic input validation (prevents letters in the input field)
